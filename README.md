# 视频管理台 Android

这是一个本地安卓视频管理应用工程，围绕以下功能实现：

- 十星评分
- 名称、评分、时长、修改时间、手动顺序等多种排序
- 手动排序下长按拖拽
- 选择文件夹后自动扫描并导入视频
- 统一卡片化展示和视频缩略图预览

## 运行与打包

1. 用 Android Studio 打开 `video-manager-android`
2. 等待 Gradle 同步依赖
3. 运行到手机，或执行 `assembleDebug` 打出 APK
