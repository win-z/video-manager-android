# 视频管理台 Android

这是一个本地安卓视频管理应用工程，围绕以下功能实现：

- 十星评分
- 名称、评分、时长、修改时间、手动顺序等多种排序
- 手动排序下长按拖拽
- 选择文件夹后自动扫描并导入视频
- 统一卡片化展示和视频缩略图预览

## 运行与打包

1. 用 Android Studio 打开 `video-manager-android`
2. 等待 Gradle 同步依赖
3. 运行到手机，或执行 `assembleDebug` 打出 APK

## 最近修复

- 修复 `MainActivity.kt` 中注释包含 `video/*` 导致 Kotlin 误判为未闭合块注释、进而让 GitHub Actions 构建失败的问题
- 清理 `MainActivity.kt` 中未使用的 import，减少编译干扰
- 收紧顶部工具栏，只保留导入、清空、网格/列表切换和网格大小切换相关控件
- 支持解析文件名前缀 `+x.xx` 与 `【x.xxx】`，并为未带分数前缀的视频默认回填 5 星
- 修复重命名后列表中文件名与相对路径不同步的问题
- 打开视频时附带 URI 读权限与 `ClipData`，提升调用系统播放器的兼容性
- 网格与列表卡片改为极简展示：正方形封面，下面只展示名字与完整评分，去掉元信息与拖拽提示
- 修复旧缓存 `0 星 / 0 分` 覆盖真实文件名前缀评分的问题，重新导入或重开应用时会优先按文件名回填评分
- 修复 `+x.xx` 前缀到十星评分的换算逻辑，改为按有效档位四舍五入，避免 `+4.75` 这类值被压低
- 展示排序默认锁定为按实际分数从高到低，同星级时再按 `4.999 > 4.998` 这种命名分值降序排列
- 列表模式继续支持长按拖拽，拖拽结束后会自动重算分数、批量重命名，并回到高分在前的展示顺序
- 网格与列表卡片补充小号数值分数显示，同时修复网格封面容器高度异常，避免部分设备上缩略图布局出错
- 卡片标题改为只展示纯标题，不再把 `【分数】` 或 `+分数` 前缀混进标题里，标题字号也进一步收小
- 新增长按编辑弹窗，可在同一个小窗口里直接改标题和评分，保存后统一触发分数重算与文件重命名
- 修复 MediaStore 快速导入后相对路径记录不正确导致 SAF 回退重命名失败的问题，提升实际文件改名成功率
